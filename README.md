# 2D-Dungeon-Crawler

## A **2D top-down dungeon crawler** built with **Unity** (C#)

A **2D top-down dungeon crawler** built with **Unity**. Explore procedurally generated rooms, fight diverse enemies, collect loot, and improve your character as you progress through challenging dungeons. Inspired by games like *Binding of Isaac* and *Enter the Gungeon*.

---

## Features

- **Procedurally generated dungeons** with multiple rooms per level
- **Player mechanics**:
  - Health, damage, and invincibility frames
  - Movement and shooting
  - Dashing
  - Animations for all player actions
- **Enemies** with diverse behaviors and attack patterns
- **Weapons and ammo** system
- **Items and loot**:
  - Chest rooms
  - Shop system to purchase weapons and items
  - Breakable objects
- **Multiple room types** and layouts for variety

---

## How to Run

1. Download the project
2. Extract the folder.  
3. Double-click the file `NEW dungeon 2021.exe` to play.
